This repository is an attempt to provide helpful, supportive and supplementary information for:
Incomplete Nature: How Mind Emerged from Matter, a book by Terence Deacon.

Publication date and details for the book:
November 21, 2011 | ISBN-10: 0393049914 | ISBN-13: 978-0393049916 | Edition: 1

Various sources for obtaining or viewing the book are available via: Wikipedia Book Sources

All the data and code in this web site is available via: Absence GitHub Dashboard

All text and code for this web site is available under an MIT License.

This repository often refers to the book simply as Incomplete Nature.

This repository has no connection with the author or publisher.
